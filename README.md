# CAPRA-ROBOT

CAPRA-ROBOT is a software suite for probabilistic catastrophe risk assessment.

The platform provides computational tools for hazard, exposure, vulnerability, and catastrophe risk modelling.

## Download

The latest official version of CAPRA-ROBOT is available from the Releases section of this repository.
